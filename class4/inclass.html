In-class Exercises

Part 1: Filtering data

  1. Download today's data.
  2. Open the streets shapefile in QGIS. These are the streets around Pratt, so you can open them in a project from the previous class if you like.
  3. Filter on RW_TYPE to only show paths and trails.
  4. Filter on RW_TYPE to show everything except paths and trails.
  5. Clear the filters to show all features.

Part 2: Quantitative marker styles

  1. Open the NYPD Motor Vehicle Collisions CSV in QGIS.
  2. Style the points using a Graduated style using the NUMBER OF PERSONS INJURED column.
  3. Experiment with styling by colors and size, try other classification modes.

Part 3: Counting points in polygons

  1. Open the Council District Boundaries shapefile.
  2. Re-open the NYPD Motor Vehicle Collisions CSV if you closed it.
  3. Find the count of points in by city council district: in the bottom left search bar in QGIS, search for Count Points in Polygon.
  4. Open the attribute table for the new layer. It should have a new column named NUMPOINTS.
  5. Keep the data open for the next part.

Part 4: Styling choropleths

  1. Continue with the Council Districts from the previous part.
  2. Using the data from the previous part, style it using a Graduated style with the new NUMPOINTS column.
  3. Try a few classification schemes and look at the histogram for each.
  4. Keep the data open for the next part.

Part 5: Normalizing counts

  1. In your project properties, change General > Units for area measurement to Square miles.
  2. In the Layers Panel, select the Council Districts layer that contains the count of collisions. Open the Field Calculator.
  3. Create a new field with name area_sqmi, type Decimal number (real), length 15, and precision 2. Use the expression $area to calculate the area.
  4. Confirm that the field was added to the attribute table.
  5. Add another field to the layer, this time dividing the number of collisions by the area of the feature in square miles. This is the density of collisions per square mile. The expression should be something like "NUMPOINTS" / "area_sqmi".
  6. Style the Council Districts using a Graduated style and your new density column. Compare the results to a Graduated style using the count column.
