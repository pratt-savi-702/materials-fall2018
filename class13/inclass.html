In-class Exercises

Part 1: QuickOSM

  1. Install the QuickOSM QGIS plugin.
  2. Zoom into a relatively small area of NYC or another city.
  3. Use QuickOSM to download OSM data tagged such that the key is highway, value is bus_stop.
  4. Look for some other tag (key-value pair) to download data from OSM for. You can use OpenStreetMap with the Map Data layer turned on, the OSM wiki, or Taginfo to help find interesting tags.

Part 2: Time Manager

  1. Download today's data.
  2. Open the earthquakes shapefile in QGIS.
  3. Install the TimeManager QGIS plugin.
  4. Use TimeManager to animate the earthquakes using the time column with a time frame of one day.

Part 3: 3D maps

  1. Open the buildings-lower-manhattan shapefile in QGIS.
  2. Add a 3d view: View > New 3D Map View.
  3. Enable 3D rendering on the buildings layer and set the extrusion. Hold shift and drag the mouse cursor to change the perspective on the 3d view.

Part 4: Data Plotly

  1. Open the queens-tenancy-income shapefile in QGIS.
  2. Install the Data Plotly plugin.
  3. Open Data Plotly: Plugins > Data Plotly > Data Plotly plugin.
  4. Make a scatterplot of income versus owners.
  5. Try the box select tool on the plot.
  6. Experiment with some other plots, such as histograms.

Part 5: qgis2web

  1. Create a simple map in QGIS using some of your data or the data included in today's data.
  2. Install the qgis2web plugin.
  3. Use qgis2web to create a webmap of your QGIS map.
  4. Create a login at neocities.org.
  5. Drag and drop all of the files generated by qgis2web into your site files at neocities and view the resulting site.

Part 6: Python

  1. Open the queens-tenancy-income shapefile in QGIS.
  2. Open the Python Console (Plugins > Python Console).
  3. With the queens-tenancy-income layer selected in the layers panel, type: 
    layer = iface.activeLayer()
    in the Python Console and press [enter].
  4. Get the count of features: 
    layer.featureCount()
  5. Get all the values in the household column: 
    for feature in layer.getFeatures():
    print(feature['households'])
  6. Run a buffer: 
    import processing
processing.runAndLoadResults('native:buffer', { 'INPUT': layer, 'DISTANCE': 100, 'SEGMENTS': 10, 'DISSOLVE': True, 'OUTPUT': 'memory:' })
