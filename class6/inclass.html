In-class Exercises

Part 1: Creating a join-able field

  1. Download this energy benchmarking dataset.
  2. Download MapPLUTO for a single borough of your choice. If you already have MapPLUTO for a borough on your computer you can skip this step.
  3. Open both in QGIS. In the energy benchmarking dataset, use the Field Calculator and the Borough, Block, and Lot fields to create a single BBL field.
  4. Create an attribute join on the MapPLUTO file you have to the energy benchmarking dataset.

Part 2: Converting joined data fields

  1. Continue with the joined data from the previous part.
  2. Select only the features where 2014 Score * isn't null.
  3. Export the selected features to a new shapefile.
  4. Select only the features where 2014 Score is "Not Available".
  5. Using the Field Calculator, update 2014 Score to be NULL for the select features.
  6. Deselect the features.
  7. Using the Field Calculator, make a new field based on 2014 Score that is an integer field.
  8. Make a graduated style based on your new field.

Part 3: Spatial Joins, summarize points on polygons

  1. Download today's data.
  2. Open City Council Districts and airbnb-listings-20181107.
  3. Use Join attributes by location (summary) from the search bar in the bottom left of the window.
  4. Do the join with these settings:
    1. Make the council districts the Input layer.
    2. Make the Airbnb listings the Join layer.
    3. For Fields to summarize, select price and calculated_host_listings_count.
    4. For Summaries to calculate, select min, max, median, and mean.
  5. Examine the resulting layer's attribute table.
  6. Style the data using a graduated style on a few of the summary fields.

Part 4: Spatial Join, add polygon data to points

This time we will add data to the Airbnb listings by which council district each falls into.

  1. Open City Council Districts and airbnb-listings-20181107 if they're not already open.
  2. Use Join attributes by location from the search bar in the bottom left of the window.
  3. Do the join with these settings:
    1. Make the Airbnb listings the Input layer.
    2. Make the council districts the Join layer.
    3. Under Join type select Take attributes of the first located feature only.
  4. Examine the resulting shapefile's attribute table.
  5. Style the resulting shapefile. For example, use a categorized style on council district number.

Part 5: More expressions

  1. Open ne_10m_admin_0_countries_lakes.
  2. Use Select by expression to try to select the following countries:
    1. ECONOMY is any of the emerging regions.
    2. LASTCENSUS is before 2000.
    3. SUBREGION is any of Caribbean, Central America, or South America with ECONOMY of developing region.
  3. Experiment with a few more expressions.
