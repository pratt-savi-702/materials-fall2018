In-class Exercises

Part 1: Styling with QGIS

  1. Download tonight's data.
  2. Open all of the data in the part1 folder in QGIS. This data is clipped to the area in Brooklyn around Pratt.
  3. Keeping the cartographic principles we talked about in mind, experiment with styling each of the layers. Add labels as necessary.
  4. Create a PDF of your map using a Print Layout.

Part 2: Opening CSVs

  1. Go to Layer > Data Source Manager.
  2. Select the Delimited Text tab.
  3. Browse for the file part2/311_noise_complaints_summer_2018.csv.
  4. Select CSV for the file format, ensure Longitude and Latitude are selected for the X and Y fields, and EPSG:4326 is selected for the CRS.
  5. Click Add to add the data to your map.

Part 3: Project Projections

  1. Open the data in the part3 folder.
  2. Go to Project > Project Properties.
  3. Select the CRS tab if it's not already selected.
  4. In the Filter text field, type 4326 and select WGS 84.
  5. Press OK.
  6. Try it again with 2263 and another projection or two.
