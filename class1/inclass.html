In-class Exercises

Part 1: Latitude and Longitude

Let's plot some points!

  1. Download today's data.
  2. First time:
    1. Visit dwtkns.com/pointplotter.
    2. Open volcanoes.txt from today's data in a text editor.
    3. Copy and paste the coordinates for the volcanoes into the text box on dwtkns.com/pointplotter and press Plot points.
  3. Second time:
    1. Visit darrinward.com/lat-long.
    2. Uncheck (Demo) Road Trip.
    3. Paste the volcanoes coordinates under Quick Lat/Lng Entry and click Add To Map.
    4. The results might be different from what you expected: why?

Part 2: Install QGIS

Windows

  1. Visit the QGIS download page.
  2. Select the appropriate package (32 bit or 64 bit) and download it.
  3. Once the package has downloaded, run it.
  4. Open QGIS to confirm that it works.

OS X

  1. Install Python 3.6.5.
  2. Visit the QGIS download page.
  3. Download the installer package for Version 3.2.
  4. Install GDAL (2 Install GDAL Complete) in the QGIS installer package.
  5. Install QGIS (3 Install QGIS 3) in the QGIS installer package.
  6. If OS X gives you an error while installing any of the packages, force it to install by right-clicking on the installer, selecting Open, and choose to install it.
  7. Open QGIS to confirm that it works.

Part 3: Load Data in QGIS

Practice opening data in QGIS and getting around the interface.

  1. From today's data, open ne_110m_admin_1_states_provinces_lakes by dragging and dropping it into QGIS.
  2. Use the hand tool (Pan Map) to move around, the zoom tools to zoom in and out.
  3. Open the attribute table by right-clicking on the layer in the Layers Panel and clicking Open Attribute Table.
  4. Do the same with nybb.geojson.
