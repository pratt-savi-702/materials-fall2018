In-class Exercises

Part 1: Clipping and Buffering

  1. In QGIS, open TRI_2017_NY.csv in today's data. You can learn more about it at the EPA's download page for the data.
  2. Also open the nybb (borough boundaries) shapefile.
  3. Under Vector > Geoprocessing Tools > Clip, clip the TRI data to the borough boundaries. The result should be a new shapefile with only TRIs in the five boroughs.
  4. Reproject the clipped version of TRI_2017_NY to 2263 so we know the units will be feet. Either right-click the layer and export with the correct CRS or use Vector > Data Management Tools > Reproject layer.
  5. Now buffer the data from the previous step by 1000 feet (Vector > Geoprocessing Tools > Buffer). Do not check Dissolve result.
  6. Now add the Public_School_Locations layer to QGIS.
  7. Select by location (Vector > Research > Select by Location) to find public schools within 1000 feet of a TRI feature.

Part 2: Overlay Operations

  1. Use the Seasnakes data included in the zipfile you downloaded. The data is a small extract of the sea snakes data available at IUCN.
  2. Use Vector > Geoprocessing Tools > Intersect to find only the places where the habitats of snake1 and snake2 overlap.
  3. Use Vector > Geoprocessing Tools > Union to find places where the habitats of snake1 and snake3 overlap and the places where each exists on its own. Style each feature differently using rule-based styles.

Part 3: Geocoding

  1. Open Queens_Library_Branches.csv in Excel.
  2. Install the MMQGIS QGIS plugin if you have not.
  3. Geocode the spreadsheet in QGIS by going to MMQGIS > Geocode > Geocode CSV with Google / OpenStreetMap. For Web Service, use OpenStreetMap / Nominatim. Don't forget to pick a place to save your shapefile and "not found" spreadsheet.
  4. Geocode the spreadsheet using the Census batch geocoder. You will want to edit the spreadsheet before uploading it to the Census geocoder so it matches the format given by the sample CSV.
